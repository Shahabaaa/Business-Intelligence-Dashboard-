# Business Intelligence Dashboard

This is a self-contained Power BI-style dashboard prototype for KPI monitoring and business performance analysis.

Open `index.html` in a browser to use it. The dashboard includes:

- KPI cards for revenue, gross margin, SLA compliance, and open risks
- Region, department, reporting-period, and target filters
- Revenue trend, revenue-by-region, and department performance charts
- Operational metrics table with status labels
- CSV export for filtered records
- A generated management summary for the current dashboard view

The sample data is generated in the page script so the project runs without installing dependencies.
